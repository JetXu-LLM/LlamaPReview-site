Frequently Asked Questions

Q: How do I install LlamaPReview?

A: You can install LlamaPReview directly from the GitHub Marketplace. Simply visit our GitHub App page and click on the "Install" button.

Q: Is there a free tier available?

A: Yes, we currently offer a free tier with usage limits. You can install and use LlamaPReview without binding any payment method.

Q: How does LlamaPReview work?

A: LlamaPReview automatically analyzes new pull requests using advanced Large Language Models. It provides detailed, AI-generated review comments to streamline your review process and improve code quality.

Q: Can I customize the review rules?

A: Yes, LlamaPReview offers customizable review rules that can be adapted to your team's specific coding standards and practices.
